The difference between padding, margin and border is that

Padding refers to the space between the element content and the element border

Margin is used to define the space around elements.It clears an area around the element.

Border is used to set border on an element. The CSS border properties (border-style, border-color, border-width and border-radius) are used to specify the style, color and size of the border of an element
